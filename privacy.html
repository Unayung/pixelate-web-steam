PIXELATE TD is committed to protecting your privacy. We collect minimal data necessary for gameplay and never sell or share your personal information with third parties.

1. Information We Collect

1.1 Data We DO NOT Collect

PIXELATE TD does not collect:

  • Personal identification information (name, email, phone number)
  • Location data or GPS coordinates
  • Photos, camera, or microphone access
  • Contact list or address book
  • Payment information (the game is completely free)
  • Social media profiles or login credentials

1.2 Data We Collect for Gameplay

To provide multiplayer functionality, we temporarily store:

  • Session Data: Temporary connection identifiers for multiplayer matches
  • Game State: Tower positions, wave progress, and gold amounts during active matches
  • Match Information: Player numbers (Player 1 or Player 2) and match status

This data is stored in memory only during active gameplay and is automatically deleted when matches end.

1.3 Technical Data

  • Device Information: Device type and OS version for compatibility
  • Game Performance: Crash reports and error logs to improve stability

2. How We Use Your Information

The limited data we collect is used exclusively to:

  • Enable real-time multiplayer matches between players
  • Synchronize game state during VS mode
  • Ensure fair gameplay and prevent cheating
  • Fix bugs and improve game performance
  • Provide technical support when you contact us

3. Multiplayer & Network Communications

3.1 Server Connection

When you play VS Mode, the game connects to our multiplayer server hosted on Render.com at:

https://pixelate-48p5.onrender.com

3.2 Data Transmission

  • All multiplayer data is transmitted over secure HTTPS/WSS connections
  • Data is encrypted in transit using industry-standard TLS/SSL protocols
  • No persistent player profiles or accounts are created
  • Match data is deleted immediately after games end

3.3 Lobby System

When creating or joining multiplayer lobbies:

  • Optional player names are stored temporarily (only during the match)
  • Lobby codes are randomly generated and expire after use
  • No chat history or messages are stored

4. Data Storage & Retention

4.1 Server Storage

  • Active Matches: Data stored in server memory during gameplay
  • Retention Period: Automatically deleted when matches end or players disconnect
  • No Long-term Storage: We do not maintain databases of player information

4.2 Local Storage

  • Game Settings: Sound preferences, language selection stored locally on your device
  • Single-Player Progress: High scores and wave records stored locally
  • Your Control: Can be cleared by deleting the app

5. Third-Party Services

5.1 Hosting Provider

Render.com - Our multiplayer server is hosted on Render's infrastructure.

  • Purpose: Provide multiplayer game server hosting
  • Data Shared: Temporary match data and connection information

5.3 No Additional Analytics

We do NOT use:

  • Google Analytics, Firebase Analytics, or similar tracking services
  • Social media SDKs or sharing tools
  • Crash reporting services that collect personal data

6. Children's Privacy

PIXELATE TD is suitable for all ages. We do not knowingly collect personal information from children under 13. The game:

  • Does not require account creation or age verification
